# TravelPin：生命周期式旅行记录助手 —— 产品定义及设计方案 📍

> **版本**：v1.1 (2026-04-05)  
> **核心哲学**：将冰冷的数据点转化为温暖的叙事海报，让每一场旅行都拥有一份“数字精装集”。

---

## 1. 产品愿景 (Vision)
TravelPin 不仅仅是一个简单的打卡工具，它是一个**全生命周期**的旅行伙伴。从最初的“灵感萌芽”，到“详细规划”，再到“身在旅途”，最后是“归来后的记忆复盘”，TravelPin 致力于在每一个触点提供仪式感与极致的审美体验。

### 核心亮点
- **审美优先**：拒绝平庸的平铺设计，采用 **Liquid Glass (流体玻璃)** 拟物化风格。
- **离线安全**：通过瓦片缓存技术（Full Interactive Offline Map），确保在无信号的秘境也能精准定位。
- **智能叙事**：集成博主风格 AI，将分散的打卡点一键编织成极具诗意的文学随笔。

---

## 2. 核心功能路线图 (Roadmap)

### 第一阶段：足迹版图 (Completed) 🏗️
- ✅ **SwiftData 持久化**：毫秒级的数据存取，支持多媒体数据的外部存储。
- ✅ **多图记录**：支持高清原图的无缝打卡。
- ✅ **足迹回顾**：维度的统计看板（行程、景点、照片数据）。

### 第二阶段：智行交互 (In Progress) 🚀
- ✅ **路线优化**：基于地理编码的每日行进轨迹动态渲染。
- ✅ **全交互离线地图**：目的地瓦片图层的预下载与平滑切换。
- ✅ **文学助手 (AI Memoir)**：多预设风格（文艺、纪实、社交媒体）的内容生成逻辑。
- ⏳ **动态动效 (Liquid Animation)**：进一步打磨 Glassmorphism 的实时光影与物理碰撞。

### 第三阶段：社交与共鸣 (Planned) 🔮
- 📅 **同行协作**：支持多人共同维护同一份“旅行手推车”。
- 📅 **灵感广场**：基于个人品味的“不期而遇”式景点推荐。
- 📅 **实体导出**：将 AI 游记直接一键排版，支持打印为实体画册。

## 3. 设计语言与规范 (Design System)

### 3.1 Vibe: Liquid Glass
我们定义了一种名为 **Liquid Glass** 的视觉风格：
- **层级 (Z-Index)**：通过背景模糊 (Material Blur) 与阴影营造通透的物理层级。
- **色彩 (TP-Palette)**：
  - `tpAccent`: 极光绿 (#00FFAB) / 晚霞紫 (#9D50BB) 的渐变。
  - `tpSurface`: 带有 0.1 饱和度的毛玻璃纹理。
- **字体 (Editorial Type)**：标题使用具有衬线感的现代字体，正文使用圆润的 Apple System Font。

## 4. 用户痛点与解决方案

| 用户痛点 | TravelPin 解决方案 | 状态 |
| :--- | :--- | :--- |
| **旅行后整理照片太麻烦** | **AI 游记** + **Trip Poster**：一键生成排版精美的叙事内容。 | ✅ |
| **偏远地区无信号，地图变空白** | **交互式瓦片缓存**：预先下载目的地地图，确保探索不间断。 | ✅ |
| **多个打卡点无法连成线** | **Route-Polyline**：自动根据时间线连接足迹，形成轨迹图。 | ✅ |
| **打包行李总会漏掉东西** | **Packing Matrix**：分品类管理的智能行李矩阵。 | ✅ |
